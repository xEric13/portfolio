Rationale

The Alumni Portal System is a web-based system intended for the alumni to update their status and to be informed on upcoming events. The system can display a reminder or an announcement on the homepage of the website. One of the functions of the alumni account is that they can update their status and current job anytime. This allows the alumnus to see how their peers have progressed professionally. The alumni can create their own account and have it visible to their fellow alumni. In this system, we provide security for the data of the alumni. All the data of the college graduates will be stored in the database server. Only the alumni and faculty can access it. The Alumni Portal System will track records of the alumni.

Objectives of the Study

This study aims to develop Alumni Portal

Specifically, the study aimed to:

1.Create alumni Account
2.Manage Alumni Account
3.Generate Alumni Status and Report
4.Send updates to alumni
5.Create a Graph Analysis to provide data for the school
6.To evaluate the system using ISO 25010

Database Architecture

Alumni can log in to the system and update their information via the web-based frontend. Admins have access to a separate admin site, which includes powerful searching and filtering tools, as well as features for importing and exporting data. The backend handles client requests that arrive through the web-based frontend, bridging the gap between client activities and the database. The Alumni Database maintains all information and data for alumni who have registered with the system.
